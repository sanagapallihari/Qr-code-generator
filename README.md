# QR Code Generator

A simple and fast QR Code Generator built with HTML, CSS, and JavaScript.

## Features
- Generate QR codes for any text or URL
- Download QR Code as PNG image
- Auto-preview while typing
- Fully responsive UI

## How to Run
Just open `index.html` in any browser OR host on GitHub Pages.

## GitHub Pages Deployment
1. Upload all files to a GitHub repo
2. Go to Settings → Pages
3. Set source: `main` branch → `/root`
4. Open the published URL

Enjoy!
